# 📘 Assignment: Python Lists

## 🎯 Objective

Practice creating, accessing, and modifying lists in Python to store and manage sequences of data.

## 📝 Tasks

### 🛠️ Create a List

#### Description
Create a list of your favorite foods and print it.

#### Requirements
Completed program should:

- Define a list called `favorite_foods` with at least 3 food items.
- Print the entire list.
- Example output: `['pizza', 'burger', 'pasta']`


### 🛠️ Access List Elements

#### Description
Access specific elements from the list using indices.

#### Requirements
Completed program should:

- Print the first food in the list.
- Print the last food in the list.
- Print the second food in the list.
- Example output:
  First food: pizza
  Last food: pasta
  Second food: burger


### 🛠️ Modify the List

#### Description
Add new foods, remove existing ones, and change items in the list.

#### Requirements
Completed program should:

- Add a new food to the end of the list.
- Remove the first food from the list.
- Change the second food to something new.
- Print the updated list after each modification.
